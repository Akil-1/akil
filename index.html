comando básico

apt

O comando apt no Linux é uma ferramenta de gerenciamento de pacotes

get

O termo "get" no contexto do Linux pode se referir ao método de requisição HTTP GET, utilizado para solicitar dados de um servidor web.

pwd

O comando pwd no Linux exibe o caminho completo do diretório atual no terminal, ajudando os usuários a identificarem rapidamente sua localização no sistema de arquivos.

comando avançado

rm -r

O comando rm -r no Linux é usado para remover diretórios e seu conteúdo de forma recursiva.
